 <h2>Statement</h2>
<div>「Technetope」は、テクノロジーによって構築された新しい“生態系”である。
2.4m 四方の空間の中を、30台の小型可動スピーカーが群れとして動く。スピーカーは音を発しながら互いの位置と人間の座標を参照し、群れを形成・分離・再統合する。その動きは、湖面を泳ぐメダカ、草原を渡るイナゴ、空を渡る鴨、夜空の星のように、環境の変化に応答し続ける。

この作品では、観客は外部の観察者ではなく、環境の一部として存在する。
カメラによって観測され、データ化された身体は、同時にスピーカー群の動態を変化させる要素となる。
人間は制御される存在でありながら、同時に自由に空間を漂う浮き草である。
観察する側と観察される側、制御する主体と制御される客体、その境界が絶えず揺らぎ続ける。

「Technetope」は、生き物のように振る舞う非生命体を通じて、
“テクノロジーが生態となる瞬間”を描き出す試みである。
それは自然と人工の二項対立を越え、
人間・機械・環境が相互に生成し合う新たな生命観の原型を提示する。</div>
